Setup Steps

  2. Create an account (free trial available)
  3. Click "Manage API keys" → "Create API key"
  4. Copy your API key (starts with "xai-")
  5. In Replit: Go to Secrets tab → Add key: XAI_API_KEY
  6. Paste your API key as the value
