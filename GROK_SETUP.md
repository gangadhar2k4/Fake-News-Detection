# Setting Up Grok AI for News Verification

## Quick Start Guide

### Step 1: Get Your Grok API Key
1. Go to https://console.x.ai/
2. Sign up with your email or Google/Twitter account
3. Click "Manage API keys" 
4. Click "Create API key"
5. Name it "News Verification" 
6. Copy your API key (starts with xai-...)

### Step 2: Add API Key to Your Project
1. In the Replit Secrets tab (left sidebar), add:
   - Key: `XAI_API_KEY`
   - Value: Your copied API key

### Step 3: Test the Integration
1. Go to the News Checker verification page
2. Paste any news article
3. Click "Verify News"
4. You'll see Grok AI analysis with detailed insights

## What You Get with Grok AI

✓ **Smart Analysis**: Advanced fact-checking beyond simple keywords
✓ **Credibility Scoring**: Confidence levels for each prediction  
✓ **Issue Detection**: Identifies specific problems in articles
✓ **Real Analysis**: Actual AI reasoning instead of demo mode

## Demo Mode vs Real API

**Without API Key (Demo Mode):**
- Basic keyword matching
- Simple fake/true detection
- Limited analysis

**With Grok API Key:**
- Advanced language understanding
- Detailed credibility assessment
- Specific issue identification
- Professional-grade fact-checking

## Troubleshooting

**"Demo mode" message:** API key not set correctly in Secrets
**Error messages:** Check API key format (should start with "xai-")
**No response:** Verify internet connection and API key validity
